# ProWindows Tracker

Business management app for ProWindows window cleaning.

## Features
- Revenue tracking (YTD, monthly, job history)
- Miles & expense logging with IRS mileage auto-conversion
- Customer roster with recurring plan tracking
- Seasonal scheduling with revenue forecasting

## Deploy to Railway
1. Push this repo to GitHub
2. Connect to Railway — it auto-deploys
3. Open the Railway URL on your phone and add to home screen

## Run locally
```
npm install
npm start
```
Open http://localhost:3000
